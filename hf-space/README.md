# Lore backend

Decision-memory API for engineering teams. Ask `/why`, capture via `/inscribe`
and the GitHub webhook. See the main repo for the full project.

Secrets to set in **Settings → Secrets**: `GROQ_API_KEY`, `DATABASE_URL`,
`GITHUB_WEBHOOK_SECRET`.
